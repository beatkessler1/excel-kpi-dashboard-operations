# Excel KPI Dashboard

## Overview
This project presents an Excel-based dashboard developed as part of a Bachelor project. It transforms structured data into clear KPIs and visual insights.

## Dataset
The dataset used in this project is provided in `dataset.xlsx`.

## Dashboard
The dashboard is available in `dashboard.xlsx` and includes:
- KPI tracking
- Charts for data visualization
- Interactive filtering using slicers

## Features
- Data aggregation using pivot tables
- Visualization with charts (bar, column)
- Interactive filtering
- KPI-based analysis

## Tech Stack
- Microsoft Excel
- Pivot Tables & Charts
- Slicers
